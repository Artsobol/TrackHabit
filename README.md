# TrackHabit - Telegram‑бот для трекинга привычек

TrackHabit - проект Telegram‑бота, который помогает формировать устойчивые привычки и отслеживать прогресс. Архитектура проекта строится как **модульный монолит** с принципами **Clean Architecture**, чтобы обеспечить масштабируемость, тестируемость и прозрачные границы ответственности.

Статус: **MVP в разработке**.

## Цели проекта

- Упростить регулярный трекинг привычек через Telegram‑интерфейс.
- Дать понятную статистику по выполнению.
- Сохранить гибкость для роста функциональности без архитектурного долга.

## Архитектура

Проект следует принципам Clean Architecture:
- **Domain**: бизнес‑сущности и правила.
- **Application**: сценарии использования и оркестрация.
- **Infrastructure**: БД, внешние интеграции, Telegram‑API.
- **Interfaces**: REST/бот‑адаптеры и DTO.

Монолит остаётся **модульным**: каждый модуль инкапсулирует собственный функционал и зависимости.

## Модули

- `common` - общие типы, базовые абстракции и утилиты.
- `users` - пользовательский контур (учёт пользователей и связанные сценарии).

## Технологии

- **Java 21**
- **Spring Boot 4**
- **Spring Data JPA**
- **Liquibase**
- **PostgreSQL**
- **MapStruct**, **Lombok**
- **Maven (multi‑module)**

## MVP (план)

Минимальный функциональный срез ориентирован на базовый трекинг привычек:
- регистрация пользователя (по Telegram ID);
- создание и управление привычками;
- отметка выполнения;
- базовая статистика.
